# MMB Website System

A real-time Sales Order (SO) and Purchase Order (PO) management system with inventory tracking, invoicing, and reporting.

## Project Structure

- **frontend/**: React (Vite) application styled with TailwindCSS and Lucide Icons.
- **backend/**: Node.js (Express) server with Prisma ORM and SQLite/PostgreSQL.

## Getting Started

1. Install dependencies in both folders:
   ```bash
   cd backend && npm install
   cd ../frontend && npm install
   ```
2. Start the development servers:
   ```bash
   # In backend/
   npm run dev
   # In frontend/
   npm run dev
   ```
